# README

### 怎么运行

- clone 项目

  ```shell
  git clone https://github.com/huangzhongxin/live-project.git
  ```

- 进入 目录

  ```shell
  cd live-project/front-end
  ```

- 安装依赖

  ```shell
  npm install
  ```

- 启动服务

  ```shell
  npm run dev:h5
  ```

- 访问 [127.0.0.1:10086](http://127.0.0.1:10086)

  如果出现界面，说明OK
